# There is no blanket disallow here and there should never be one. Blocking the
# root path for every user-agent removes the site from search entirely.

# Worth knowing: this file is not the whole story. A Render Free web service
# answers /robots.txt itself with a blanket block whenever the instance is spun
# down, and that response never reaches server.mjs. See the hosting note in
# README.md. If the site is de-indexed, check the host before you edit this file.
